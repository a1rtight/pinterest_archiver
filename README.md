# Pinterest Archiver

Open source image downloader for Pinterest. Save boards to your downloads instantly, no strings attached.

## Installation

1. Visit [a1rtight.github.io/pinterest_archiver](https://a1rtight.github.io/pinterest_archiver/)
2. Drag the bookmarklet to your bookmarks bar

## Usage

1. Go to any Pinterest board
2. Click "Pinterest Archiver" in your bookmarks bar
3. Select sections to download
4. Click Start

Images download as a ZIP file with folders for each section.

## Features

- Downloads full resolution images
- Supports board sections
- Streaming downloads for large boards (6000+ pins)
- Pause and resume functionality
- No extension or signup required
